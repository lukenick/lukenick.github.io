Version 0.2 - Pushing pudding increments counter

Now that we have our elements on the page we can start adding some interactivity. We’re going to begin by building a MVP of the game - in digital technology an MVP is a Minimum Viable Product. What’s the smallest version of your project you can make that still has some point.

In this case I think the minimum viable product is when you push the image the counter goes up.

Now in our javascript the first thing we need to do is say that we start off with zero puddings, so let's go ahead and make a variable that says that - check out Lines 11.

Next we want to create an EventListener to listen out for when someone pushes our pudding. When they do, it runs the function pushPudding() which does two things. One - it increments the amount of puddings (increment means increase by one). Two - it then updates the counter on screen to show the new value of our variable.

Version 0.4 - Adding percentage upgrades

Time to make something to do with all the puddings we've been pushing. It’s time for our first upgrade. A core mechanic of this style of game is that when you accumulate enough currency (ie puddings) you can use those puddings to purchase an upgrade which helps you earn more currency (puddings) in the future. The simplest upgrade to program is a percentage upgrade. What that means is when you purchase this upgrade you no longer get one pudding each time you press, but you now get slightly more.
